# ООО "Обувь" - Система управления продажами

[![Maintainability](https://api.codeclimate.com/v1/badges/placeholder/maintainability)](https://codeclimate.com)

## Описание проекта

Веб-приложение для управления продажами обуви, разработанное в рамках практического задания.

## Функциональность

### Модуль 1: База данных
- Создана база данных в 3 нормальной форме
- Таблицы: пользователи, товары, заказы, пункты выдачи, категории, производители, поставщики
- Импорт данных из Excel файлов

### Модуль 2: Авторизация и просмотр товаров
- Окно входа с аутентификацией
- Ролевая система: Гость, Авторизированный клиент, Менеджер, Администратор
- Просмотр товаров с подсветкой:
  - Скидка > 15%: зеленый фон (#2E8B57)
  - Товар отсутствует на складе: голубой фон
  - Перечеркнутая цена при скидке

### Модуль 3: CRUD операции с товарами
- Добавление, редактирование, удаление товаров (только для Администратора)
- Поиск по текстовым полям
- Фильтрация по поставщику
- Сортировка по количеству на складе
- Валидация форм

### Модуль 4: Работа с заказами
- Просмотр списка заказов
- Добавление, редактирование, удаление заказов (только для Администратора)
- Добавление товаров в заказ
- Расчет общей суммы заказа

## Технологии

- React 18
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- ESLint

## Установка и запуск

```bash
# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev

# Сборка для production
npm run build

# Проверка линтером
npm run lint
```
