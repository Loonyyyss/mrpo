# ООО "Обувь" - Система управления продажами (Python Desktop)

[![Maintainability](https://api.codeclimate.com/v1/badges/placeholder/maintainability)](https://codeclimate.com)

## Описание

Десктопное приложение для управления продажами обуви, написанное на Python с использованием Tkinter.

## Требования

- Python 3.8+
- Tkinter (обычно входит в стандартную поставку Python)

## Установка и запуск

```bash
# Перейти в папку проекта
cd obuv_app

# Запуск приложения
python main.py
```

## Функциональность

### Модуль 1: База данных
- SQLite база данных в 3 нормальной форме
- Таблицы: users, products, orders, categories, manufacturers, suppliers, pickup_points, order_items
- Предзаполненные данные из Excel-файлов

### Модуль 2: Авторизация и просмотр товаров
- Окно входа с аутентификацией
- Ролевая система
- Просмотр товаров с подсветкой:
  - Скидка > 15%: зеленый фон (#2E8B57)
  - Товар отсутствует: голубой фон
  - Перечеркнутая цена при скидке

### Модуль 3: CRUD товаров
- Добавление, редактирование, удаление товаров (Администратор)
- Поиск по текстовым полям
- Фильтрация по поставщику
- Сортировка по количеству

### Модуль 4: Работа с заказами
- Просмотр списка заказов
- Добавление, редактирование, удаление заказов
- Добавление товаров в заказ
- Расчет общей суммы

```
├── database/
│   ├── __init__.py
│   └── db.py          # Работа с SQLite
├── views/
│   ├── __init__.py

│   ├── main_window.py     # Главное окно (товары)
│   ├── product_dialog.py  # Диалог товара
│   ├── orders_window.py   # Окно заказов
│   └── order_dialog.py    # Диалог заказа
├── utils/
│   ├── __init__.py
│   └── constants.py   # Константы приложения
├── assets/            # Ресурсы (фото товаров)
├── main.py           # Точка входа
└── requirements.txt  # Зависимости
```
